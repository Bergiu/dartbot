Ein Telegram-Bot der erfolgreiches Dartspiel mit 1 Woche Auszeit belohnt

## Wie kann ich ihn benutzen?

1. Leg deinen telegram bot API token in die Wurzel des Quellcode-Verzeichnisses als `bot_token.txt`
2. `cargo run`
